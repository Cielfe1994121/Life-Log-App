# Life-Log-App (Prototype v1.0)

日々の出来事や思考をターミナルから素早く記録し、後から自在に検索するための最小単位（MVP）プロトタイプです。
「人生の可視化」を目指すプロジェクトの第一歩として、シンプルかつ堅牢な設計を目指しました。
まだまだ至らぬ点ございますが、大目に見ていただけると幸いです。

## 🛠 特徴
- **高速入力**: 起動してテキストを打ち込むだけで、正確なタイムスタンプと共に保存されます。
- **二段構えの検索機能**: 
  1. 日付指定検索（例：`2025-12-25`）
  2. キーワード検索（「だし巻き卵」などの単語によるあいまい検索）
- **ポータブル設計**: 実行ファイルと同じディレクトリに `life_log.db` を自動生成するため、環境を選ばず動作します。
- **入力ガード**: 削除時のID指定に数値チェックを導入するなど、初心者の操作ミスによるエラーを防止しています。

## 📂 技術構成
- **Language**: Python 3.10+
- **Database**: SQLite3 (標準ライブラリ)
- **Environment**: WSL2 (Ubuntu), Windows 11

## 🚀 使い方
1. 本リポジトリをクローンするか、`life_log.py` をダウンロードします。
2. ターミナルで `python life_log.py` を実行します。
3. プロンプトに従って記録を入力してください。終了は `exit` です。

## 📝 開発の背景
このアプリは「人生可視化アプリ」の最小構成プロトタイプとして作成されました。
現在はCLI（コマンドライン）形式ですが、今後はデータの視覚化機能やGUIの導入を検討しています。
